# Website UI kit — Pontotoc County Democrats

A click‑through prototype of the county party's public website. Single‑file React (Babel‑in‑browser).

Sections:
- Header / nav with logo lockup
- Hero with horizon stripe + star row
- Upcoming events (RSVP toggles)
- Candidate spotlight
- Get involved form (volunteer / donate / register)
- Footer with meeting time + address

Open `index.html` in a browser.
